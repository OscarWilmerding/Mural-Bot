The user is building a robot muralist project. They are the sole developer across two PlatformIO codebases (base module and chassis) as well as custom PCB design (KiCad) and CAD (SolidWorks). They make frequent small targeted changes across both codebases and want their assistant to be fluent in the full codebase so changes can be located and executed quickly without needing to re-explain context each time.

The user is hands-on and technical — they work at the embedded C++ / Arduino level, design their own PCBs, and model their own hardware. Frame answers at that level.
